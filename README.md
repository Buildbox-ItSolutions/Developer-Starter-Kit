# Developer-Starter-Kit

This document describes what has to be installed, and configured in your local machine so you can work on tipical Buildbox development projects.

---

## 1- Create a user as: name@buildbox.com.br

(This step shall be done by a Buildbox administrator)

---

## 2- Create the following bookmarks in your Browser:

2.1- BBox Calendar (https://calendar.google.com/calendar/r)

2.2- Email: name@buildbox.com.br

2.3- BBox Jira (https://buildbox.atlassian.net/jira/your-work)

2.4- Trello (save each project link)

2.5- Bitbucket (https://bitbucket.org/dashboard/overview)

2.6- BBox GitHub (https://github.com/Buildbox-ItSolutions)

2.7- BBox Google Drive (https://drive.google.com/drive/u/0/my-drive)

2.8- Tracker Buildbox (https://tracker.buildbox.com.br/)


Request access, when needed

---

## 3- Install:

Frontend

3.1- VSCode: https://code.visualstudio.com/

3.2- Install GitKraken: https://www.gitkraken.com/

3.3- Install node.js: https://nodejs.org/en/


Backend

3.4- Install robo3T: https://robomongo.org/

3.5- Install Postman: https://www.postman.com/downloads/


---

## 4- Configure:

4.1- VSCode with the following plugins: TSLint, ESLint, Prettier

4.2- GitKraken -> Clone a repo, and Open a repo

4.3- robo3T -> Connect to specific project
